Researchers Verification Tool

Upload the Researchers Evaluation Analysis file and the two verification files from the system (Research Scientist and Research Technologist). The tool will process the following sheets automatically: All Researchers, Research Scientist, and Research Technologist. It uses the KFUPM ID as the key and Category/Rank or Sheet Name to determine which verification file to use. Ensure the column names are consistent: 'Name', 'Research Center', 'Final Rating', and 'Positivity Score'.
